A small independent cinema company is going to provide a self-service ticketing kiosk at one of their cinemas in London. Customers can purchase film tickets on the day using the kiosk. Kiosk software, in most cases, needs to be tailored to each kiosk solution. The task is to develop the software for the first generation of the kiosk. Your task now is to develop the software for the first generation of the kiosk. 

Compile&Run:
input these commands in cmd:
javac -d bin -cp src;lib/junit-4.12.jar;lib/hamcrest-core-1.3.jar src/*.java
java -cp bin Start


Test:
input these commands in cmd:
java -cp bin;lib/junit-4.12.jar;lib/hamcrest-core-1.3.jar org.junit.runner.JUnitCore TestFilm
